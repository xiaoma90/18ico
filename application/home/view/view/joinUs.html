加入我们

小娜 Nara
  • 2017年07月30日上午03:49
  • 更新于
关注

18ico作为国内最资深的ico众筹平台之一，为了给广大用户提供更好的体验，现需要招聘运营经理2名、客服专员5-10名。

把握区块链行业核心创业机会，亲身感受中国区块链行业爆发式的增长。

工作坐标：北京、深圳。薪资丰厚，发展空间广阔。

如有意愿，请将简历作为附件发送至：hr@18ico.com

运营经理（2名）

岗位职责：

  1. 全面负责公司运营部署与管理工作，含平台业务运营、日常维护、活动策划、社区运营等，搭建用户运营体系，根据用户生命周期及投资偏好细分用户，建立运营指标；

  2. 负责用户活跃、留存、转化目标，依据用户需求、制定运营方案或运营机制；

  3. 保持资产规模、用户数量，调整运营策略，并有效对流失用户进行预警、防流失和唤醒，保持用户价值提升；

  4. 能够独立策划线上活动、书写文案并协调上线，以及活动期间的过程管理和活动结束后的效果分析；能有效推进各类用户运营策略联动，并在项目和活动结束后对相关信息和数据进行收集和整理，分析效果，提出改善计划；

  5. 协助部门监测和研究市场动态及竞争对手策略，制定针对性策略和运营方案；

  6. 负责部门的人员管理，包含：业务技能培训、人才梯队建设等。

任职要求：

  1. 本科及以上学历，3年以上专职用户运营经验，有团队管理经验者优先；

  2. 具备良好的人际关系沟通协调能力和书面表达能力， 超强的语言表达能力和判断力；

  3. 有科学的互联网用户认知，懂得用户分级和金字塔模型，并懂得不同类型用户的运营方法；

  4. 有良好的用户运营方法论，擅长通过群组、活动等和用户互动的方式，对不同梯度的用户进行运营。有独立线上下活动策划执行能力；

  5. 有较好的数据分析能力，并能熟练应用各类办公软件；

  6. 熟悉区块链、社区、金融或互联网金融优先，18ico用户优先。

运营专员（5-10名）

岗位职责：

  1. 通过QQ群、微信群等工具为客户提供满意细致的服务；

  2. 负责微博等社交新媒体的管理；

  3. 负责热线电话的接听，业务咨询、信息查询及疑难问题的解答工作；

  4. 受理客户的投诉、建议及意见等，做好相关记录及反馈工作，及时跟进处理结果；

  5. 负责用户引导，产品使用过程中问题的解答和培训，产品更新后及时通知到客户；

  6. 主动宣传平台最新咨询动态，做好客户关系维护工作。

任职要求：

  1. 大专及以上学历，年龄20-30岁，接受应届毕业生；

  2. 普通话标准，有良好的沟通能力及语言表达能力；

  3. 头脑清晰，思路敏捷，较强的执行力，工作积极、耐心细致，具备高度的责任感；

  4. 具有较强的组织协调能力、优秀的沟通和谈判技巧、良好的团队合作精神；

  5. 熟悉办公软件和网络工具；

  6. 熟悉区块链、社区、金融或互联网金融优先，18ico用户优先；

  7. 有客服呼叫相关工作经验、有出色英语书面和语言沟通能力优先。
